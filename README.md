## 📱 项目简介

**SnapStraight（随手正一正）** 是一款专注于图片矫正的极简移动应用。通过智能边缘检测和透视矫正技术，一键将倾斜、歪斜的证件、票据、笔记等照片自动拉直、裁剪和优化。

### ✨ 核心特点

- 🎯 **单一功能**：专注图片矫正，无冗余功能
- ⚡ **极速处理**：单张图片处理时间 < 1秒
- 📦 **超轻量**：安装包大小 ≤ 2MB
- 🔒 **隐私安全**：本地处理，不联网，不上传
- 🌍 **多语言**：支持8种语言
- 🆓 **完全免费**：无广告、无内购

### 🎬 使用场景

- 👴 老人拍社保卡、医保卡报销
- 👶 宝妈拍疫苗本、出生证明存档
- 💼 上班族拍报销发票、合同扫描
- 📚 学生拍课堂笔记、错题本整理

- 📸 **AVFoundation**：相机功能

1. **边缘检测** → 智能识别文档四个角点
2. **透视变换** → 自动拉直倾斜角度
3. **智能裁剪** → 去除多余背景
4. **自动提亮** → 优化暗光照片

# 一、项目概述

## 1.1 项目名称

「随手正一正」（Snap Straight）—— 单功能极简证件/票据/笔记一键矫正裁剪APP。英文名中“Snap”对应“随手”，体现快速便捷；“Straight”对应“正一正”，直指矫正核心，简洁易记且贴合功能。

## 1.2 项目背景

日常生活中，用户使用手机拍摄身份证、社保卡、报销发票、课堂笔记、宝宝疫苗本等高频物品时，常因手持角度偏差、拍摄环境复杂等问题，出现“拍歪、拍斜、背景杂乱、光线不足”等情况。系统自带相册的编辑功能，需手动拖动角度滑块、框选裁剪区域，操作繁琐且精准度低，中老年群体、儿童及不熟悉手机操作的用户往往难以驾驭；现有第三方图片工具多功能冗余，集成滤镜、美颜、批量处理、云存储等非核心功能，不仅干扰核心需求，还普遍存在广告弹窗频繁、需强制联网、安装包体积过大等问题，严重影响使用体验。

本项目聚焦“全民高频刚需+零门槛操作”的核心诉求，摒弃所有非必要功能，专门打造仅含“一键自动矫正+裁边”的单功能工具，同时新增多国语言支持，覆盖全球更多用户，精准解决所有人群拍图后“快速变正、变清晰、易存档”的核心痛点，有效填补市场上极简型图片矫正工具的空白。

## 1.3 项目目标

1. 打造一款“体积最小、操作最简单、识别最精准”的单功能图片矫正工具，全面覆盖全年龄段用户，重点适配中老年群体与学生群体的使用习惯。

2. 核心量化指标：操作步骤严格控制≤2步、核心场景识别准确率≥98%、APP安装包体积≤2M、全程离线可用、无任何广告与内购付费模块。

3. 成为用户手机中“高频刚需、用完即走”的必备轻量工具，凭借极致体验与多国语言支持，提升全球用户留存率，依托口碑传播实现自然增长。

## 1.4 核心价值

- 极致便捷：仅需2步操作即可完成全流程，系统自动识别物体边缘、拉直倾斜角度、裁剪多余背景，无需任何手动调整，真正实现零学习成本。

- 全民适配：精准覆盖多场景需求，老人拍社保卡报销、宝妈拍疫苗本存档、上班族拍发票提交审核、学生拍笔记复习，全人群、全场景无缝适配。

- 隐私安全：所有图片均在手机本地完成处理，不联网、不上传至任何服务器，从根源上保护证件、合同等敏感信息不泄露。

- 轻量化适配：安装包体积小、运行时不占用过多内存、功耗低，完美适配所有安卓/iOS机型（包括配置较低的老年机与入门级手机）；新增多国语言支持，不额外增加体积负担。

# 三、核心功能设计

## 3.1 功能总览

APP仅搭载1个核心功能模块（自动矫正+裁边），无任何冗余附加功能，仅配套自动提亮、本地保存2项基础适配功能，最大限度压缩体积与运行负荷，确保极致轻量化与流畅性。

## 3.2 核心功能：一键自动矫正+裁边

### 3.2.1 操作流程（严格控制2步，无额外步骤）

1. 第一步：选图/拍照（首页仅设置2个超大功能按钮，无其他任何选项，避免视觉干扰）
       

    - 「拍照」按钮：点击后快速启动相机，默认开启自动对焦与自然光模式，无变焦、闪光灯、滤镜等参数调节选项，拍后直接自动进入矫正环节，无中间加载页面。

    - 「选图」按钮：点击后调用手机本地相册，仅展示图片文件，自动过滤视频、音频等非图片格式，避免用户误选，选中目标图片后自动进入矫正环节，无需手动确认。

2. 第二步：自动处理+保存
        

    - 自动处理：1秒内完成“边缘智能识别→倾斜角度拉直→多余背景精准裁剪”全流程，同时针对暗光、逆光等复杂光线环境拍摄的图片，自动优化亮度与对比度，不改变原图色彩基调，仅确保文字、图案清晰可辨。

    - 保存操作：矫正完成后直接展示处理后的预览图，底部仅设置1个“保存”超大按钮，点击后快速保存至手机相册指定文件夹，无“分享、编辑、删除”等额外选项，保存成功后自动返回首页，方便用户进行下一次操作。

### 3.2.2 核心技术点（确保识别精准度与处理效率）

- 边缘识别算法：优化轻量级识别模型，可精准适配矩形证件（身份证、社保卡）、不规则票据（褶皱发票、单据）、笔记（横线/竖线边界、无规则纸张）等多种场景，自动区分物体主体与背景，即使背景存在花纹、褶皱等干扰元素，也能精准锁定主体轮廓。

- 角度自动校准：支持±45°范围内的任意角度倾斜矫正，通过算法快速计算最优水平/垂直角度，自动拉直图片，矫正后无明显变形，确保物体比例协调。

- 智能裁边：基于边缘识别结果，自动裁剪掉桌面、墙面、手部等多余背景，仅保留物体主体部分，裁边误差≤1mm，视觉上无明显偏差，同时避免裁剪到主体内容。

## 3.3 配套适配功能（不增加操作复杂度，默认开启）

- 自动提亮：针对室内、阴天、逆光等光线不足场景拍摄的图片，自动优化亮度与对比度，提升文字、图案的清晰度，同时严格控制提亮幅度，避免过度曝光导致细节丢失。

- 本地离线处理：所有图片的识别、矫正、裁剪、保存均在手机本地完成，不联网、不上传至服务器，不生成任何日志文件，从根源上保护证件、合同等敏感信息的隐私安全。

- 误触防护：选图/拍照后不设置“取消”按钮，简化操作流程，若用户无需保存处理后的图片，直接按手机系统返回键即可退出，返回首页，不生成临时文件，不占用内存。

## 3.4 功能边界（明确不做的功能，坚守单功能定位）

不做滤镜、美颜、水印、文字添加、图片旋转等编辑功能；不做批量处理、图片分享、云存储、文件管理等扩展功能；不做注册、登录、会员体系、积分兑换等账号相关功能；不做开屏广告、弹窗广告、信息流广告等任何广告推送与弹窗提醒功能。

# 四、UI/UX设计

## 4.1 设计原则

- 极简无冗余：每个页面的视觉元素严格控制≤5个，无多余图标、文字、过渡动画，避免视觉干扰，让用户快速聚焦核心功能。

- 全民适配：采用超大功能按钮、超大显示字体、高对比度色彩搭配，充分适配中老年群体的视力特点与操作习惯，同时兼顾儿童与中青年用户的使用体验。

- 一致性统一：全APP设计风格、操作逻辑保持高度一致，无多级跳转层级，所有操作仅包含“点击”和“返回”两个动作，降低用户记忆成本。

## 4.2 页面设计（仅3个页面，无二级菜单）

### 4.2.1 首页（核心操作页）
